# 変数と値

変数と値のパーツは、状態を覚えたり、条件の材料を作ったりするためのパーツです。

## 名前の注意

同じ `action` や `score` という名前でも、種類が違うと別物です。

| ブロック | 何を覚えるか | 例 |
|---|---|---|
| `フラグ open` | ON/OFF | ドアが開いているか |
| `数値変数 score` | 数字 | サイコロの出目、点数 |
| `文字変数 message` | 文字 | 表示したい文章 |
| `クールダウン action` | 残り秒数 | 連打防止の待ち時間 |

たとえば `クールダウン action` と `文字変数 action` は、同じ名前でも別の箱です。

初心者向けの目安:

- 点数やサイコロの目 → `数値変数`
- 表示する文章 → `文字変数`
- ON/OFFだけ → `フラグ`
- 何秒待つか → `クールダウン`

## フラグ NAME

名前つきのtrue/falseを覚えます。

使いどころ:

- ドアが開いているか
- 罠が起動済みか
- スイッチが押されたか

## フラグ NAME を true/false/切り替え にする

フラグの値を変更します。

使いどころ:

- 押したら `open` をtrueにする
- もう一度押したら開閉を切り替える
- 一度だけ起動した記録を残す

## 数値変数 NAME

名前つきの数値を覚えます。

使いどころ:

- スコア
- ランダム結果
- 受け取った数値
- カウント

## 数値変数 NAME を N だけ増やす

数値変数に指定した数を足します。

使いどころ:

- 得点を増やす
- 使用回数を数える
- ヒット数を増やす

## 文字変数 NAME

名前つきの文字を覚えます。

使いどころ:

- 受け取った文字
- 表示メッセージ
- 状態名

## 真偽値変数 NAME / true / false

trueまたはfalseを扱います。

使いどころ:

- 表示ON/OFF
- 判定結果の保存
- 複数条件の整理

## 数値 N

そのままの数値です。

使いどころ:

- 座標
- 速度
- 範囲
- 秒数

## 計算

数値同士を足す、引く、掛ける、割ることができます。

使いどころ:

- 速度を調整する
- 受け取った値を加工する
- ランダム結果から別の値を作る

## A から B までのランダム数

指定した範囲のランダムな数を作ります。

使いどころ:

- サイコロ
- 当たり判定
- ランダム移動

## クールダウン NAME 中 / 残り秒数 / 開始する

名前つきの残り時間を管理します。

使いどころ:

- 連打防止
- 再使用待ち
- 一定時間ごとの処理

注意:

- 同じ名前を使うと同じクールダウンを共有します。
- 名前が違うと別のクールダウンとして扱われます。
