# 通信

通信パーツは、別のアイテムへ合図を送るためのパーツです。

Jcreate++では、アイテム同士の連動を作るときに通信を使います。

## 近くのアイテムへ送信

指定した範囲内のアイテムへメッセージを送ります。

使いどころ:

- 周囲のライトを一斉に点ける
- 近くのギミックをまとめて起動する
- エリアスイッチ

注意:

- 範囲が広すぎると大量送信になることがあります。
- `once` やクールダウンと組み合わせると安全です。

## 指定アイテムへ送信

World Item Reference Listに登録した名前のアイテムへメッセージを送ります。

使いどころ:

- スイッチからドアへ送る
- ボタンから表示板へ送る
- 中央制御アイテムへ送る

Unity側の準備:

- 送信先アイテムをWorld Item Reference Listに登録します。
- Jcreate++側の名前とUnity側の参照名を一致させます。

## 値つきメッセージ

メッセージと一緒に、数値、文字、真偽値、ハンドルを送ります。

使いどころ:

- 点数を送る
- ランダム結果を送る
- 開く/閉じるをtrue/falseで送る
- 衝突相手を別アイテムへ渡す

## 受け取った値

メッセージ受信イベントの中で、受け取った値を使います。

使いどころ:

- 受け取った数値を変数に保存する
- 受け取った文字を表示する
- 受け取った真偽値で表示を切り替える

注意:

- 受信値は `メッセージを受け取ったとき` の中だけで使えます。
- 一連の動作であとから使いたい場合は、受信直後に変数へ保存します。

## 送ってきた相手へ返信

メッセージを送ってきた相手に返信します。

使いどころ:

- 確認応答
- スイッチへ成功/失敗を返す
- 呼び出し元だけ反応させる

注意:

- 受信イベントの中だけで使います。

## 衝突相手へ送信

衝突した相手にメッセージを送ります。

使いどころ:

- ぶつかった相手だけ動かす
- 当たったアイテムだけ音を鳴らす
- ゴールに触れたプレイヤーやアイテムへ合図する

注意:

- `衝突したとき` の中だけで使います。
- 衝突相手が送信可能なハンドルでない場合は送信されません。
