# 制御と待機

制御パーツは「条件によって動きを変える」「順番に処理する」ためのパーツです。

## もし

条件がtrueのときだけ中のブロックを実行します。

使いどころ:

- フラグが立っているときだけ動かす
- 乱数が一定以上なら当たりにする
- 範囲内にプレイヤーがいるときだけ反応する

## もし / でなければ

条件がtrueなら上、falseなら下の処理を実行します。

使いどころ:

- 当たりとはずれ
- 開いている/閉じている
- クールダウン中/使用可能

## once制御

条件がfalseからtrueに変わった瞬間だけ、中のブロックを1回実行します。

使いどころ:

- 範囲に入った瞬間だけ音を鳴らす
- スイッチが押された瞬間だけ送信する
- 衝突した瞬間だけ得点する

注意:

- 条件がtrueのままだと2回目は動きません。
- もう一度動かすには、条件が一度falseに戻る必要があります。

## 一連の動作

中の処理を順番に進めます。

使いどころ:

- 開く、待つ、閉じる
- 音を鳴らす、文字を変える、待つ、戻す
- 演出を段階的に動かす

注意:

- `毎フレーム` の中には置けません。
- 待機ブロックは一連の動作の中で使います。

## 秒待つ

指定した秒数だけ待ってから次へ進みます。

使いどころ:

- ドアを数秒開けたままにする
- 音のあとに表示を変える
- 演出の間を作る

## 条件まで待つ

条件がtrueになるまで待ちます。

使いどころ:

- フラグが立つまで待つ
- プレイヤーが近づくまで待つ
- クールダウンが終わるまで待つ

## 指定秒数、毎フレーム実行

指定した秒数の間、中の処理を毎フレーム実行します。

使いどころ:

- 数秒だけ動かす
- 数秒だけ回転させる
- 短い演出を作る
