# 動作

動作パーツは、アイテムやプレイヤーの位置、回転、速度を変えるためのパーツです。

## 位置を指定する

アイテムの位置を指定した座標にします。

使いどころ:

- 初期位置へ戻す
- ドアを開いた位置へ移動する
- ゴールに着いたら場所を変える

注意:

- 大きすぎる値を入れると見えない場所へ移動します。
- 動かすアイテムはScriptable Itemとして動かせる設定が必要です。

## 位置を少し変える

今の位置から、指定した分だけ移動します。

使いどころ:

- 押すたびに少し動く
- 毎フレームでゆっくり進む
- ランダム移動の補助

## 位置を秒数つきで動かす

現在位置から、指定した差分だけ、指定秒数をかけて滑らかに移動します。

例:

- `位置を x:0 y:2 z:0 へ 1 秒かけて動かす`

使いどころ:

- ドアをゆっくり開ける
- 足場を少しずつ動かす
- メッセージを受け取ったときに滑らかに移動する

注意:

- `毎フレーム` の中では使いません。
- インタラクト時やメッセージ受信時など、「動き始めるタイミング」に置きます。
- 指定する x / y / z は「最終的にどれだけ動くか」です。毎フレームの移動量ではありません。

## 角度を指定する / 角度を少し変える

アイテムの向きを変えます。

使いどころ:

- 看板を正面へ向ける
- 回転スイッチ
- 動く床の向きを変える

## 角度を秒数つきで回す

現在の角度から、指定した差分だけ、指定秒数をかけて滑らかに回転します。

例:

- `角度を x:0 y:0 z:90 へ 1 秒かけて回す`

使いどころ:

- ドアをゆっくり回して開ける
- レバーを倒す
- 看板やギミックを演出つきで回す

注意:

- `毎フレーム` の中では使いません。
- インタラクト時やメッセージ受信時など、「回り始めるタイミング」に置きます。
- 指定する x / y / z は「最終的にどれだけ回すか」です。毎フレームの回転量ではありません。

## 往復する

指定した軸方向へ、一定幅で往復します。

置ける場所:

- `毎フレーム`

使いどころ:

- 移動床
- 浮遊する足場
- 動く障害物

## 回転し続ける

指定した軸で回転し続けます。

置ける場所:

- `毎フレーム`

使いどころ:

- 回転床
- 風車
- 障害物

## 一定時間移動して戻る

指定方向へ一定時間動き、その後もとの位置へ戻します。

置ける場所:

- `毎フレーム`

使いどころ:

- 一定時間だけ開くドア
- せり上がる足場
- 押し出しギミック

## ランダムワープ

現在位置を中心に、指定範囲へランダムに移動します。

使いどころ:

- ランダム宝箱
- ワープ床
- 動き回る的

## 力を加える

アイテムに瞬間的な力を加えます。

使いどころ:

- 跳ねる物体
- 押し出される箱
- 衝突リアクション

注意:

- 物理の影響を受けるアイテムでないと見た目に出ないことがあります。

## プレイヤー速度 / ジャンプ速度を変える

イベントに関わったプレイヤーの速度倍率を変えます。

置ける場所:

- `インタラクト時`
- `持ったとき`
- `離したとき`

注意:

- 一連の動作の中では使えません。
- 速度を戻す処理も用意しておくと安全です。
