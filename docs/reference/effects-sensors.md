# 演出と検知

演出と検知のパーツは、見た目や音を変えたり、近くの状況を調べたりするためのパーツです。

## 音を鳴らす

Item Audio Set Listに登録した音を再生します。

使いどころ:

- ボタン音
- 当たり音
- 衝突音
- 開閉音

Unity側の準備:

- アイテムのItem Audio Set Listに音を登録します。
- Jcreate++側の音IDとUnity側のIDを一致させます。

注意:

- 毎フレームの中にそのまま置くと連続再生になります。
- `once` やクールダウンと組み合わせるのがおすすめです。

## サブノードの文字を変える

指定したサブノードのText/TextMesh/TextMeshPro/TextMeshProUGUIの文字を変えます。

使いどころ:

- スコア表示
- 結果表示
- 受付メッセージ
- 状態表示

Unity側の準備:

- 指定した名前のサブノードを用意します。
- そのサブノードに対応するText系コンポーネントを付けます。

注意:

- サブノード名とコンポーネント種類が一致しないと反映されません。

## サブノードの表示/有効を切り替える

指定したサブノードのUnityコンポーネントの `enabled` をtrue/falseにします。

使いどころ:

- MeshRendererをfalseにして見た目を消す
- Lightをtrueにしてランプを点ける
- Colliderをfalseにして当たり判定を消す
- TextMeshProをfalseにして文字を隠す

対応コンポーネント:

- MeshRenderer
- SkinnedMeshRenderer
- Light
- BoxCollider
- SphereCollider
- CapsuleCollider
- MeshCollider
- Text
- TextMeshPro
- TextMeshProUGUI

注意:

- TextMeshは `enabled` の対象ではないため、このパーツでは選べません。
- Animatorや別ギミックが同じプロパティを触っている場合、動きが競合することがあります。

## 範囲内プレイヤー人数

このアイテムの近くにいるプレイヤー人数を返します。

使いどころ:

- 人数で開くドア
- 近くに人が多いほど表示を変える
- エリア内の人数カウント

## 範囲内にプレイヤーがいる

指定範囲内にプレイヤーが1人以上いるかをtrue/falseで返します。

使いどころ:

- 自動ドア
- 近づくと点灯するランプ
- 範囲センサー

注意:

- 毎フレームで使う場合、範囲を必要以上に広くしないようにします。

## 前方に何かがある

アイテムの前方向へraycastし、指定距離内に何かがあるかを判定します。

判定対象:

- 何か
- アイテム
- プレイヤー

使いどころ:

- 前方センサー
- 向いている先にあるときだけ反応
- 的当て判定
- 視線方向の簡易チェック

Unity側の準備:

- 判定したい対象にRaycastで当たるColliderやShapeの設定が必要です。

注意:

- アイテムの向きが基準です。
- 判定対象のLayerやCollider設定によって、当たらないことがあります。
