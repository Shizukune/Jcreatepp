# ランダム宝箱

宝箱を調べると、ランダムで結果が変わるレシピです。

## 使うブロック

- `インタラクト時`
- `ランダム数`
- `数値変数をセット`
- `もし`
- `メッセージを送る`

## 作り方

1. 宝箱に `インタラクト時` を置きます。
2. `数値変数 result` に `1 から 3 までのランダム整数` を入れます。
3. `もし result = 1` なら、当たり演出のメッセージを送ります。
4. `もし result = 2` なら、別の演出にします。
5. `もし result = 3` なら、ハズレ演出にします。

## Unity側で必要な設定

- 演出先へ送る場合は、送信先アイテムを `World Item Reference List` に登録します。
- 音や色変更を使う場合は、それぞれのリスト設定が別途必要です。

## アレンジ

- クールダウンを入れて連打を防ぐ。
- メッセージ値で `result` の数値を別アイテムへ送る。
- 当たりのときだけドアを開く。
