# 近づくと点灯するランプ

## 何ができる？

プレイヤーが近づいたときだけ、ライトや見た目をONにします。

新しく追加された `範囲 5 m にプレイヤーがいる` と `サブノード Lamp の Light を true にする` を使うレシピです。

## 使う場面

- 近づくと点くランプ
- プレイヤー検知センサー
- 人がいるときだけ表示される案内
- 自動ドアの前段階

## 必要なUnity / Cluster設定

- スクリプトを動かすアイテムに Scriptable Item を設定します。
- ライトを切り替えたい場合は、サブノードにLightコンポーネントを付けます。
- 見た目を切り替えたい場合は、サブノードにMeshRendererを付けます。
- サブノード名を `Lamp` などにして、Jcreate++側の名前と合わせます。

## Jcreate++で置くブロック

1. `毎フレーム`
2. `もし 範囲 5 m にプレイヤーがいる なら`
3. `サブノード Lamp の Light を true にする`
4. `でなければ`
5. `サブノード Lamp の Light を false にする`

## 生成される考え方

毎フレーム、自分の位置を中心に近くのプレイヤーを探します。

範囲内に1人以上いればLightを有効にし、いなければ無効にします。

## 応用

- LightではなくMeshRendererを切り替えて、オブジェクトを出したり消したりする
- TextMeshProを切り替えて、案内文を表示する
- `once` を使って、近づいた瞬間だけ音を鳴らす
- 範囲内プレイヤー人数で、人数表示を変える

## よくあるミス

- サブノード名が違う
- Lightコンポーネントが付いていない
- 毎フレームで音を鳴らしてしまう
- 範囲が広すぎて意図しない場所でも反応する
