# レシピ一覧

Jcreate++で作れるギミックを、短いレシピとしてまとめています。

どのレシピも「何ができるか」「Unity / Cluster側で必要な設定」「Jcreate++で置くブロック」を先に見られるようにしています。

## まず試すなら

- [触ったら動く](./interact-move.md)
  - アイテムを触ると、指定した方向へ少し動かします。
- [サイコロ判定](./dice-judge.md)
  - ランダムな数値を使って、当たり・はずれのような分岐を作ります。
- [クールダウンつきボタン](./cooldown.md)
  - 連打できないボタンや、一定時間待つ仕組みを作ります。

## 読み方

最初から全部覚えなくて大丈夫です。

1. 作りたい動きに近いレシピを選ぶ
2. Unity / Cluster設定を確認する
3. Jcreate++でブロックを置く
4. 生成された考え方を読む
5. 動かなければトラブルシュートを見る

困ったら [トラブルシュート](../troubleshooting.md) も見てください。
