# 踏むと1回だけ反応する床

床にぶつかった、または踏んだタイミングで、1回だけ処理するレシピです。

## 使うブロック

- `衝突したとき`
- `もし 〜 になった瞬間だけ実行`
- `クールダウン中`
- `クールダウンを開始する`
- `メッセージを送る`

## 作り方

1. 床アイテムに `衝突したとき` を置きます。
2. 条件に `クールダウン active 中ではない` 形を作ります。
3. 中でメッセージ送信や位置変更を行います。
4. 最後に `クールダウン active を 2 秒で開始する` を置きます。

## Unity側で必要な設定

- 衝突判定用の Collider / Physical Shape が必要です。
- 相手がアイテムまたはプレイヤーでない場合、`衝突相手` は空になります。

## アレンジ

- 踏むとランプへ `on` メッセージを送る。
- 踏むと床が少し沈む。
- 踏むとトラップが2秒だけ作動する。
