# 触ったら動く

## 何ができる？

プレイヤーがアイテムを触ったときに、そのアイテムを少し動かします。

例として、スイッチを押すと足場が上がる、ボタンを押すと箱が横にずれる、といった動きに使えます。

## 使う場面

- 押すと開くドア
- 触ると上がる足場
- 説明パネルを押すと別の場所へ移動する仕掛け

## 必要なUnity / Cluster設定

- スクリプトを動かすアイテムに Scriptable Item を設定します。
- プレイヤーが触れる対象として使う場合は、Interactできる設定が必要になることがあります。
- アイテム自体を動かす場合は、移動可能な設定が必要になることがあります。

## Jcreate++で置くブロック

1. `インタラクト時`
2. 中に `位置を x:0 y:1 z:0 ずつ変える`

毎回同じ場所へ移動したい場合は、代わりに `位置を x:0 y:1 z:0 にする` を使います。

## 生成される考え方

`インタラクト時` の中で、アイテムの位置を書き換えます。

`ずつ変える` は今の位置から相対的に動かします。`にする` は指定した座標へ直接移動します。

## 注意

- 何度も触ると移動量が積み重なる設計にするか、毎回同じ場所へ移動する設計にするかを先に決めます。
- 別のアイテムを動かす場合は、今のブロックだけではなくメッセージ送信や参照設定が必要になることがあります。

## よくあるミス

- Scriptable Item が付いていない
- Interactできる対象になっていない
- `にする` と `ずつ変える` を間違える
- 移動量が大きすぎて、アイテムが見えない場所へ飛ぶ
