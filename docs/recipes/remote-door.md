# 遠隔スイッチで開くドア

## 何ができる？

スイッチを触ると、別のアイテムにメッセージを送り、ドアを開きます。

スイッチとドアを別アイテムにできるので、ワールド内の仕掛けらしい見た目にしやすいレシピです。

## 使う場面

- 壁のボタンで開くドア
- 離れた場所の橋を動かすスイッチ
- 複数の装置を連動させる入口

## 必要なUnity / Cluster設定

スイッチ側:

- Scriptable Itemを付けます。
- Interactできる設定にします。
- World Item Reference Listに、ドアを `door` などの名前で登録します。

ドア側:

- Scriptable Itemを付けます。
- 移動できる設定にします。
- `メッセージ open を受け取ったとき` を使います。

## Jcreate++で置くブロック

スイッチ側:

1. `インタラクト時`
2. `もし true になったら 指定アイテム door に メッセージ open を1回だけ送る`

ドア側:

1. `メッセージ open を受け取ったとき`
2. `位置を x:0 y:3 z:0 にする`

## 生成される考え方

スイッチ側は `door` という参照名のアイテムを探して、`open` メッセージを送ります。

ドア側は `open` を受け取ったときだけ、自分の位置を変更します。

## 応用

- ドア側で音を鳴らす
- ドア側でTextMeshProの文字を `OPEN` にする
- スイッチ側にクールダウンを付けて連打を防ぐ
- `open` と `close` の2種類のメッセージを作る

## よくあるミス

- World Item Reference Listの名前とJcreate++側の名前が違う
- 受信側のメッセージ名が `open` ではない
- ドア側アイテムにScriptable Itemが付いていない
- ドアを動かせない設定になっている
