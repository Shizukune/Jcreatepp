# スイッチでドアを開ける

ボタン役のアイテムを押すと、ドア役のアイテムへメッセージを送り、ドア側が動くレシピです。

## 使うブロック

送信側:

- `インタラクト時`
- `フラグをtrueにする`
- `指定アイテムにメッセージを1回だけ送る`

受信側:

- `メッセージを受け取ったとき`
- `位置をずつ変える`

## 作り方

1. ドアのアイテムに、受信イベントを置きます。
2. メッセージ名を `open` にします。
3. 中に「位置をずつ変える」を置きます。例: `Y: 2`
4. スイッチのアイテムに、インタラクト時イベントを置きます。
5. `open` メッセージをドアへ送ります。

## Unity側で必要な設定

- ドアを `World Item Reference List` に登録します。
- 登録名はJcreate++側の指定アイテム名と同じにします。例: `door`
- ドアをスクリプトで移動できる設定にします。

## よくある失敗

- 参照名が `door` と `Door` で違っている。
- ドア側に `open` 受信イベントがない。
- ドアが移動できない設定になっている。
