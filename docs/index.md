## まず何を作る？

最初はこの3つがおすすめです。

- [スイッチでドアを開ける](/recipes/switch-door)
- [踏むと1回だけ反応する床](/recipes/collision-pad)
- [ランダム宝箱](/recipes/random-chest)

Jcreate++は「イベント」「条件」「動作」「メッセージ」を組み合わせて使います。  
Unity側の設定が必要なブロックもあるので、先に [Clusterルール早見表](/cluster-rules) を軽く見ておくと迷いにくいです。
