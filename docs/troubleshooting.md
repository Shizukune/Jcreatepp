# トラブルシュート

## メッセージが届かない

確認すること:

- メッセージ名が完全に同じか。
- 送信先の参照名が合っているか。
- 送信先アイテムが `World Item Reference List` に登録されているか。
- 受信側に `メッセージを受け取ったとき` があるか。

## 衝突したときが動かない

確認すること:

- Collider / Physical Shape があるか。
- 衝突する相手が判定対象になっているか。
- `衝突相手` ブロックを `衝突したとき` の外で使っていないか。

## 何度も実行される

毎フレームや衝突中は、同じ処理が何度も走ることがあります。

対策:

- `もし 〜 になった瞬間だけ実行` を使う。
- `クールダウン` を入れる。
- メッセージ送信は `1回だけ送る` 系を使う。

## アイテムが動かない

確認すること:

- アイテムがスクリプトで移動できる設定になっているか。
- 移動量が小さすぎないか。
- 受信イベントや条件が実行されているか。

## 文字や真偽値がうまく送れない

値つきメッセージでは、送信側と受信側の型を合わせます。

例:

- 送信: `文字`
- 受信: `受け取った文字`

型が違う場合、Jcreate++は安全な初期値にします。
